Description of Application:

The following web app has been created by Sooruj Ghangass as part of his SE3316 coursework.


This web app allows a users to sign up so that they can create collections. Alternatively, unregistered users are able to browse public collections! There is a lot of functionality related to the collection objects that registered users can create. A registered user may rate the collection for example. While the collection owner is not allowed to rate their own collection, I have allowed them to change visibility settings if they want their collection to be private or public.
